#  Predicción y Análisis de Abandono Estudiantil (Student Dropout)

##  Sobre el Proyecto
Este proyecto tiene como objetivo principal acompañarme en el descubrimiento y mejora de mis habilidades en el análisis de datos. A través de este trabajo práctico, busco comprender el paso a paso de cómo construir un buen modelo predictivo desde cero, entendiendo el proceso detrás de los datos.

Adentrándonos en el proyecto en sí, el objetivo central es **predecir las probabilidades de que un alumno abandone o no una cursada**, logrando identificar qué variables tienen mayor impacto sobre nuestra variable a predecir.

##  Metodología del Proyecto
Para llevar a cabo este análisis, seguí un flujo de trabajo estructurado:
1. **Identificación de Datos:** Búsqueda de un dataset que cumpliera con los requerimientos necesarios para el aprendizaje.
2. **Preparación:** Análisis inicial y limpieza profunda de los datos.
3. **Análisis Exploratorio (EDA):** Visualización y comprensión del comportamiento de las variables.
4. **Transformación:** Análisis sobre la necesidad de utilizar (o no) un *encoder* para el preprocesamiento.
5. **Modelado Predictivo:** Aplicación, entrenamiento y evaluación de un modelo de Machine Learning (Árbol de Decisión).

##  Fuente de Datos

## Herramientas y Tecnologías
A lo largo de este proyecto, aprendí y utilicé el siguiente stack tecnológico:
* **Lenguaje:** Python 
* **Manipulación y Análisis de Datos:** Pandas, NumPy
* **Visualización:** Matplotlib, Seaborn
* **Machine Learning:** Scikit-Learn
* **Modelo Aplicado:** `DecisionTreeClassifier` (Árbol de Decisión)
